# StaySetu

StaySetu is a student-focused rental platform designed to help users find PGs, flats, roommates, and related services easily and efficiently.

## 🚀 Features
- Property listings (PGs, flats)
- Roommate search
- Maid services integration
- WhatsApp lead tracking
- User authentication (Appwrite / Supabase)

## 🛠 Tech Stack
- React (Vite)
- Tailwind CSS
- Appwrite / Supabase
- Vercel (deployment)

## 📂 Project Structure
- `src/` → Frontend source code
- `public/` → Static assets
- `schema.sql` → Database schema

## ⚙️ Setup
```bash
npm install
npm run dev
